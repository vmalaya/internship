 # struts2jpa
 ## Test project for studying basics ORM via JPA 
 
 ### Project contains following technologies:
 - [JBoss EAP 6.4]
 - [Struts 2 framework]
 - [CDI plugin]
 - [Convention plugin]
 - [JPA]
 - [Flyway] database migration 
 - Mysql database
